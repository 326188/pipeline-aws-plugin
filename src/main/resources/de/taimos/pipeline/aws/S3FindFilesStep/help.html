List all of the files/folders in the bucket. If path is given, then it will be used as the root of the search.

The following are equivalent:

  • s3FindFiles bucket: "my-bucket", glob: "path/to/my/file.ext"
  • s3FindFiles bucket: "my-bucket", path: "path/to", glob: "my/file.ext"
  • s3FindFiles bucket: "my-bucket", path: "path/to/", glob: "my/file.ext"

List every file:

  • s3FindFiles bucket: "my-bucket", glob: "**", onlyFiles: true
